# Interactive Quiz Challenge

A premium, modern interactive quiz application built with HTML, Tailwind CSS, and Vanilla JavaScript.

## 🚀 Features

- **Dynamic Question Loading**: Questions are loaded and rendered dynamically.
- **Real-time Scoring**: Immediate feedback on correct answers and score updates.
- **Immersive Progress Bar**: Visual feedback on quiz progression.
- **Responsive Design**: Fully responsive layout that works on mobile and desktop.
- **Premium Aesthetics**: Glassmorphism, smooth animations, and a modern color palette.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure.
- **Tailwind CSS**: Modern utility-first styling.
- **JavaScript (ES6+)**: Core logic and interactivity.
- **Google Fonts**: Plus Jakarta Sans & Manrope for premium typography.
- **Material Symbols**: Interactive iconography.
